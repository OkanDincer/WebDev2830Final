Smartly Budget:

Final Project Setup:
-Make sure you have a MongoDB Atlas account setup
-Log into MongoDB Atlas, choose a cluster and click Connect
-Make sure the .env file within server matches your MongoDB connection string 
-Make sure you have Node.js downloaded

Execution:
-Seed the database by running the command "node scripts/seed.js" in the terminal within the server folder
-Run the app using two terminals
-To launch the backend, run npm run dev within the terminal in the server folder
-To launch the frontend, run npm start within the terminal in the client folder
-Alternatively, double click on launch.bat within the root folder to run both commands simultaneously

Website:
-At localhost, port 3000, you should see the Smartly Budget application
-The dashboard will show your income (per month), expenses (per month), and your balance
-The balance is calculated as your income minus your expenses
-Add monthly income using the Add Income page
-Add monthly expenses using the Add Expense page
-The budget page allows you to add additionally budgeting options per month
